# Quick Draw 🎨

**Quick Draw** is a lightweight, reference image viewer designed for gesture drawing and art practice.

## Features
* **Flexible Sessions:** Customizable intervals from 10 seconds to 1 hour, supporting sessions up to 120 images.
* **Smart Persistence:** Automatically saves your preferred folder and session settings between runs.
* **Adaptive UI:** Interface that maintains aspect ratio and scales dynamically as you resize your window.

## Installation & Usage

### Windows & Linux
1. Download the latest `.zip` release from the [Releases page](https://github.com/DragonInstall/quick-draw/releases).
2. Extract the folder and run `Quick Draw.exe` (Windows) or the `Quick Draw` binary (Linux).

### macOS
1. Download the `QuickDraw-MacOS.zip` release.
2. Extract the folder to reveal `Quick Draw.app`.
